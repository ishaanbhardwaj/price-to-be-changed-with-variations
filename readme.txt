Show only the selected variation price in WooCommerce variable product selected.
Instead the Messy Full Range of Prices and Sales or Writing Code in function.php.


== Description ==

Replace the Variable Price range by the chosen variation price in WooCommerce

<strong>No options</strong> are in Version 1.0, just activate the plugin and it’s done.

== Installation ==

1. Go to your WP Dashboard > Plugins and search for ‘Price To Be Changed With Variations’ or…
2. Download the plugin from WP repository.
3. Upload the plugin folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
